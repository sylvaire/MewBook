# MewBook — 项目状态

## 产品

MewBook：Android 本地记账应用（Kotlin、Jetpack Compose、Room、Hilt）。核心能力包括流水、分类、账本、资产、预算、周期模板、统计下钻、WebDAV 同步、自动备份、备份导出、智能导入与应用内更新。
