# 主页周期、分支管理、预算与动效优化设计

## 目标

本次迭代覆盖 7 个改动：

1. 主页记账周期选择在下次打开时保持为上次选择的 `日 / 周 / 月`
2. 主页周期切换改为长胶囊三等分控件，与下方汇总卡片等宽
3. 分支管理中设置默认分支后不改变列表排序
4. 分支管理右侧上下箭头始终按当前可见顺序生效
5. 预算管理补全分类预算能力
6. 页面切换动效更轻、更快
7. 分支管理右下角悬浮按钮与分类管理保持一致

## 设计决策

### 主页周期持久化

- 仅持久化主页的周期类型，不持久化具体日期
- 下次进入主页时恢复上次选择的粒度
- 锚点日期仍以当天初始化，避免用户回到过期日期范围

### 主页周期控件

- 共享一个新的分段控件组件展示 `日 / 周 / 月`
- 控件为单个长胶囊，三等分布局，宽度与汇总卡片一致
- 选中态使用主色块，未选中态保持轻量背景与文本对比

### 分支排序与默认分支

- 分支列表只按手动顺序字段展示，不再把默认状态作为排序条件
- 默认分支仅影响视觉标记，不影响列表位置
- 上下箭头按照当前列表位置与相邻项交换顺序，默认分支也可参与排序

### 分类预算

- 预算页保留总预算卡
- 新增独立的“添加分类预算”入口
- 新增时从未配置预算的支出分类中选择；编辑时保留当前分类
- 分类预算进度继续按当前账本、当前周期的分类支出金额计算

### 页面切换动效

- 为导航切换补充更短时长的轻量过渡
- 顶层页面采用快速淡入淡出
- 二级页面采用短距离滑动 + 淡入淡出，退出更快于进入

### 分支管理悬浮按钮

- 改为与分类管理相同的 `FloatingActionButton`
- 统一尺寸、形状、配色和点击反馈

## 测试与验证

- 单元测试覆盖：
  - 主页周期持久化值的解析与容错
  - 分支排序逻辑不受默认状态影响
  - 分类预算可选分类过滤逻辑
- 手动验证覆盖：
  - 主页周期样式与恢复行为
  - 分支设置默认、排序、箭头点击
  - 预算新增/编辑分类预算
  - 页面切换动效节奏
