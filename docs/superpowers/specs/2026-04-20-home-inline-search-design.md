# 首页内联搜索与周期胶囊优化设计

## 目标

本次迭代覆盖 3 个点：

1. 修复首页 `日 / 周 / 月` 胶囊按钮点击时出现方形按压背景的问题
2. 调整周期胶囊与背景色阶，让选中态和容器层级更清晰
3. 为首页增加“全局搜索当前账本记录”的内联搜索能力

## 设计决策

### 周期胶囊

- 保持首页原有长胶囊三等分布局
- 分段按钮改为圆角裁切后的点击区域，避免出现方形按压背景
- 外层容器增加更明确的底色与描边，选中项保留更强的主色强调

### 首页搜索

- 搜索入口放在首页顶部栏右侧，默认展示搜索图标
- 点击后进入搜索态，顶部栏右侧切换为关闭按钮，内容区显示搜索框
- 搜索态下隐藏周期切换和汇总卡片，避免页面同时出现两种列表语义
- 搜索范围为当前默认账本下的全部记录，不受首页当前周期限制
- 搜索匹配内容包含：备注、分类名、金额字符串、账户名
- 搜索结果仍复用现有记录卡片，点击后继续进入编辑记录流程

## 验证

- 单元测试覆盖首页搜索匹配逻辑：
  - 仅搜索当前账本记录
  - 支持备注 / 分类 / 金额 / 账户名匹配
  - 空查询返回空结果
- 手动验证覆盖：
  - 周期胶囊点击不再出现方形背景
  - 胶囊底色与选中态区分更清晰
  - 搜索态切换、结果展示、结果点击编辑流程正常
